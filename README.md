
# 🌐 Wuys Edge Manager

> 🚀 An open-source tool to manage Microsoft Edge profiles, track points, and automate your workflow.
---

## 🧩 Open Source

**Wuys Edge Manager là dự án mã nguồn mở (Open Source).**

- 🔓 Bạn có thể tự do sử dụng, chỉnh sửa và phát triển thêm
- 🤝 Hoan nghênh mọi đóng góp (pull request, issue, idea)
- 💡 Mục tiêu: tạo một tool quản lý profile Edge mạnh nhưng đơn giản

---

## 📌 Giới thiệu

**Wuys Edge Manager** giúp bạn:

- 📂 Quản lý nhiều profile Microsoft Edge
- 📊 Theo dõi điểm (daily & total)
- ➕ Cộng / chỉnh sửa điểm dễ dàng
- 📋 Copy profile nhanh chóng
- 🔄 Phân loại profile tự động (Roblox / Chô li bi / Khác)
- 📅 Lưu lịch sử hoạt động

---

## 🚀 Tính năng chính

- ✅ GUI (Tkinter - Dark Mode)
- ✅ Auto detect profile từ hệ thống
- ✅ Giới hạn điểm mỗi profile (6750)
- ✅ Thống kê realtime
- ✅ Copy profile giữ nguyên dữ liệu
- ✅ Lịch sử 30 ngày

## ⚙️ Cách sử dụng

### 🔹 Dùng .exe
- Chạy `WuysEdgeManager.exe`

### 🔹 Dùng Python

```bash
python main.py
````

---

## 🛠️ Build

```bash
build.bat
```

hoặc:

```bash
pyinstaller --onefile --noconsole main.py
```

---

## ⚠️ Lưu ý

* Tool dùng thư mục:

  ```
  C:\Users\<User>\AppData\Local\Microsoft\Edge\User Data
  ```
* Nên backup trước khi dùng
